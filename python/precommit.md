#### precommit | Frecuent commands

##### Display version
```
$ pre-commit --version
```

##### Install
```
$ pip install pre-commit
```

##### Install the Hooks from ".pre-commit-config.yaml"
```
$ pre-commit install
```

##### To test the hooks run
```
$ pre-commit run --all-files
```
