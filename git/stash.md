### git | Stash

---

##### Create new stash
```
$ git tash -m "<reference>"
```
###### Example
```
$ git tash -m "001"
```

##### Apply specific stash
```
$ git stash apply <number-reference>
```
###### Example
```
$ git stash apply 1
```

##### Remove specific stash
```
$ git stash drop <number-reference>
```
###### Example
```
$ git stash drop 1
```
