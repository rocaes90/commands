# git

## Frecuent commands

#### Update branch name
How to rename a branch
```
$ git branch -m <old-name> <new-name>
```
###### Example
```
$ git branch -m feature/OP-001 feature/OP-002
```

#### Publish local repository
```
$ git push -u origin <branch-name>
```
###### Example
```
$ git push -u origin feature/OP-001
```
